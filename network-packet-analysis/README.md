# Network Packet Analysis Investigation

## Scenario
Network traffic was captured and analyzed to determine whether suspicious communication occurred between hosts on the network.

## Tools Used
Wireshark  
Network packet inspection tools  

## Investigation Process
1. Imported the packet capture (PCAP) file into Wireshark.
2. Filtered traffic by protocol, IP address, and port.
3. Examined suspicious packet streams.
4. Reconstructed communication sessions.

## Key Findings
Packet analysis revealed abnormal communication patterns between hosts. Investigation of packet streams helped reconstruct the network activity and identify suspicious connections.
