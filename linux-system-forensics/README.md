# Linux System Forensics Investigation

## Scenario
A forensic investigation was conducted on a Linux system to determine whether unauthorized activity had occurred.

## Tools Used
Linux command line tools  
Log analysis utilities  
Forensic investigation techniques

## Investigation Process
1. Examined system logs including authentication and system activity logs.
2. Investigated user accounts and login attempts.
3. Analyzed key directories such as /var/log, /etc, /bin, and /proc.
4. Reviewed command history and system artifacts.

## Key Findings
System logs and command histories provided insight into login attempts and system activity. Analysis of these artifacts allowed investigators to reconstruct events and identify suspicious behavior.

## Skills Demonstrated
Linux forensic analysis  
Log investigation  
System artifact analysis  
Timeline reconstruction
