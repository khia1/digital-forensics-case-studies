# Steganography Investigation

## Scenario
Digital image files were analyzed to determine whether hidden information had been embedded using steganography techniques.

## Tools Used
StegExpose  
OpenPuff  
Forensic image analysis tools  

## Investigation Process
1. Collected image files suspected of containing hidden data.
2. Analyzed the files using steganalysis tools.
3. Examined statistical anomalies within the image structure.
4. Attempted extraction of hidden content.

## Key Findings
Analysis revealed that hidden information had been embedded inside an image file. Extraction tools successfully recovered concealed data that contained location coordinates and additional metadata.

Steganography detection  
Digital artifact analysis  
Hidden data extraction  
Forensic investigation methodology
