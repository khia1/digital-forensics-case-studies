# Deleted File Recovery Investigation

## Scenario
A forensic investigation was conducted on disk images to recover deleted files and determine whether important evidence had been removed from the system.

## Tools Used
Autopsy  
PhotoRec  
FTK Imager  

## Investigation Process
1. Acquired disk images from the target system.
2. Loaded the images into forensic recovery tools.
3. Scanned file systems for deleted artifacts.
4. Reconstructed deleted files and examined recovered data.

## Key Findings
Multiple deleted files were successfully recovered from the disk image. These artifacts allowed investigators to reconstruct prior system activity and determine the presence of previously removed data.

## Skills Demonstrated
File system analysis  
Deleted artifact recovery  
Digital evidence reconstruction  
Forensic tool usage
